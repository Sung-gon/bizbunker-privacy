🛡️ BizBunker 개인정보처리방침

100% OFFLINE · ZERO DATA COLLECTION

본 문서는 BizBunker(이하 "앱") 이용자의 개인정보 보호에 관한 사항을 안내합니다.

1. 수집하는 개인정보 항목

본 앱은 사용자의 개인정보를 일절 수집하지 않습니다.

  • 앱 내 모든 데이터(재무 정보, 거래처 정보, 설정값 등)는 사용자의 기기 내부(로컬 SQLite 데이터베이스)에만 저장됩니다.
  • 외부 서버로 전송되는 데이터는 없습니다.
  • 분석 도구(Analytics), 광고 SDK, 추적 코드가 일절 포함되어 있지 않습니다.

2. 네트워크 통신

본 앱은 다음의 경우에만 선택적으로 네트워크를 사용합니다:

  • 환율 정보 조회: 사용자가 해외 자산 시세 조회 기능을 직접 실행할 경우, 공개 API에 시세 요청을 보냅니다. 이때 사용자의 개인정보는 전송되지 않습니다.
  • 인앱 구매: 앱 구매 및 라이선스 검증을 위해 Apple의 StoreKit 시스템과 통신합니다.

상기 기능을 사용하지 않는 경우, 앱은 완전히 오프라인으로 작동합니다.

3. 데이터 저장 및 백업

  • 모든 데이터는 기기 내 앱 샌드박스 영역에 저장됩니다.
  • 백업 기능은 기기 내 로컬 파일 시스템에 AES-256 암호화된 스냅샷을 생성합니다.
  • 백업 파일은 사용자가 직접 관리하며, 클라우드에 자동으로 업로드되지 않습니다.

4. 제3자 제공

본 앱은 사용자 데이터를 제3자에게 제공하지 않습니다.

5. 아동의 개인정보 보호

본 앱은 만 13세 미만 아동을 대상으로 하지 않으며, 아동의 개인정보를 의도적으로 수집하지 않습니다.
